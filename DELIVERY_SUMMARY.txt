================================================================================
                            QUICK START STEPS
================================================================================

1. INSTALL OLLAMA
   $ curl -fsSL https://ollama.ai/install.sh | sh
   (Or download from https://ollama.ai/)

2. PULL MODEL & START SERVER
   $ ollama pull mistral
   $ ollama serve

3. INSTALL DEPENDENCIES (in another terminal)
   $ cd ollama_research
   $ pip install -r requirements.txt

4. RUN THE SYSTEM
   $ python main.py
   OR
   $ python cli.py research "Your research question"
